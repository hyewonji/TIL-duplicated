# HTML

구조

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width" />
    <title>HTML</title>
  </head>

  <body>
    <!-- tags -->
  </body>
</html>
```

- html 안에 `head`, `body` 태그로 나누어져있다.
- head 안에 페이지에서 필요한 정보들, body안에 사용자에게 보여지는 요소들이 있다.<br>
  _html 태그 관련 자세한 내용은 MDN 사이트에 검색하면 볼 수 있다._
